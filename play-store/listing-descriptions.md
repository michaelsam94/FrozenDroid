## Full description (4000 chars max)
FrozenDroid helps power users control preinstalled and background Android packages without losing track of what is safe to touch.

Built around Shizuku-enabled package commands, FrozenDroid gives you a focused workspace for reviewing installed apps, freezing selected packages, restoring them when needed, and grouping repeat actions into reusable freeze presets.

Key features:

- Browse installed packages with search and safety labels
- Freeze and unfreeze packages individually or in selected groups
- Create freeze presets for common modes like commute, work, or battery saving
- Review an offline safety directory for manufacturer and system packages
- Add local safety overrides when you understand a package's role
- Monitor CPU wakeup telemetry to identify packages that keep waking the device
- Protect dangerous system components from accidental freeze actions
- Follow built-in Shizuku setup guidance when privileges are not ready

FrozenDroid is designed for careful, local package management. The app includes offline guidance so you can make better decisions before disabling carrier, OEM, or system packages.

Use FrozenDroid to reduce background noise, keep battery-heavy packages under control, and switch between freeze profiles with more confidence.
